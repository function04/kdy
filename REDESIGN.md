# Just For Me - UI Redesign Plan

## 현재 앱 개요

개인 일정관리 + 활동 추적 + 분석 PWA (Vite + React + Tailwind + Supabase)
모바일 중심, 매일 수회 사용, 학생 대상

## 리디자인 방향

### 디자인 원칙
1. **네이티브 앱 느낌** — 60fps, 제스처, 끊김 없는 전환
2. **정보 밀도** — 한눈에 파악, 질리지 않는 디자인
3. **간편한 조작** — 제스처, 최소 탭으로 기능 사용
4. **확장성** — 기능 추가돼도 복잡해지지 않는 구조
5. **AI 느낌 X** — 자연스럽고 의도된 디자인

### 구분감 방식 (카드 대체)

| 영역 | 방식 | 설명 |
|------|------|------|
| 상단 요약 (D-day, Upcoming) | 텍스트 + 여백 | 카드 없이 숫자와 라벨만 |
| 진행 중 활동 | Accent Bar + Elevation | 배경색 차이로 강조 |
| 할 일 목록 | Grouped List | iOS 설정앱 스타일, 들여쓰기 구분선 |
| 주간/월간 요약 | Bento Grid | 크기 차이로 중요도 표현 |
| 섹션 구분 | Frosted Sticky Header | 스크롤 시 고정 블러 헤더 |
| 설정 | Grouped List | iOS 스타일 |

### 색상 팔레트

**배경 (Elevation 시스템)**
```
--bg-base:      #0A0A0B   (앱 배경)
--bg-elevated:  #141416   (카드/섹션)
--bg-elevated2: #1C1C1F   (중첩 카드/모달)
--bg-elevated3: #242427   (호버/액티브)
```

**텍스트**
```
--text-primary:   #EDEDEF  (제목)
--text-secondary: #A0A0A8  (부제목)
--text-tertiary:  #5C5C66  (타임스탬프)
```

**테두리**
```
--border-subtle:  rgba(255,255,255,0.06)
--border-default: rgba(255,255,255,0.10)
```

**기능별 컬러 — 기존 유지 (변경 없음)**
- 활동 색상: 기존 ACTIVITY_CONFIG + COLOR_POOL 그대로
- 우선순위 색상: 그대로

### 폰트
- Pretendard Variable (한국어+영어 최적화, Inter 기반)
- CDN: `https://cdn.jsdelivr.net/gh/orioncactus/pretendard/dist/web/variable/pretendardvariable.css`

### 카드 스타일 (필요한 곳만)

**Flat Card** (일반)
```css
background: #141416;
border: 1px solid rgba(255,255,255,0.06);
border-radius: 16px;
```

**Glass Card** (히어로/강조용, 1-2개만)
```css
background: rgba(255,255,255,0.04);
backdrop-filter: blur(24px) saturate(150%);
border: 1px solid rgba(255,255,255,0.08);
border-radius: 20px;
```

**Pressable** (터치 피드백)
```css
transition: 150ms ease;
:active { transform: scale(0.97); background: #1C1C1F; }
```

└──────────────┘ └──────┘ └──────┘
┌──────┐ ┌─────────────────────┐

- D-day/Upcoming/Today → 카드 제거, 한 줄 텍스트

- Recent Task → Grouped List (iOS 스타일)
- 제스처 유지: 더블탭 완료/종료

### 일정 페이지 (/schedule)

**변경 사항:**
- 캘린더/주간 스트립 → 기존 유지 (잘 동작함)
- 태스크 리스트 → Grouped List 스타일로 변경
- MemoCard → 왼쪽 컬러 바 + 깔끔한 행 레이아웃
- 스와이프 제스처 → 기존 유지
- MemoForm → 기존 유지 (자연어 파싱 잘 됨)
- FAB → 기존 유지

### 분석 페이지 (/analytics)

**변경 사항:**
- 탭 버튼 → 기존 글래스 탭 유지
- 통계 행 → Divider + 여백 스타일
- 바 차트 → 기존 유지 (잘 동작함)
- 히트맵 → 기존 유지
- 카드 래핑 → Elevation 배경으로 변경

### 설정 페이지 (/settings)

**변경 사항:**
- 전체 → iOS Grouped List 스타일
- 섹션별 둥근 그룹 + 들여쓰기 구분선

### 활동 패널

**변경 사항:**
- 패널 글래스 효과 → 기존 유지 (잘 동작함)
- 버튼 그리드 → 기존 유지
- 애니메이션 → 기존 유지

## 구현 순서

### Phase 1: 기반 (globals.css + tailwind.config)
1. 색상 팔레트 적용
2. Pretendard 폰트 적용
3. 카드/배경 스타일 교체
4. 터치 피드백 (scale 0.97) 기본 적용
5. 노이즈 텍스처 유지

### Phase 2: 대시보드 리디자인
1. 상단 요약 → 텍스트 행
2. 활동 트래커 → Accent Bar
3. Recent Task → Grouped List
4. 모든 제스처/인터랙션 연동 확인

### Phase 3: 일정 + 설정 페이지
1. 태스크 리스트 → Grouped List
2. 설정 → iOS Grouped List
3. 스와이프/폼 기존 동작 확인

### Phase 4: 분석 페이지
1. 카드 → Elevation 배경
2. 통계 행 스타일 정리
3. 차트/히트맵 기존 유지

### Phase 5: 마이크로 인터랙션
1. 페이지 전환 개선
2. 스켈레톤 로딩
3. 부드러운 리스트 애니메이션
